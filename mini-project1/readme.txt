To run the python file:
- Create a folder called in 'input'
- Put the images in the input-folder
	- An example of a mask and an input image is already placed in the input folder
- Run the script
	- Give the script a file name for an image, ex: "DJI_0237.JPG"
	- Give the scricpt a file name for a mask, ex: "DJI_0237_masked.JPG"
- Be aware that segementing in rgb colour distance will take some time(up 10 min)
- See the results in the terminal and the output folder created by the script
